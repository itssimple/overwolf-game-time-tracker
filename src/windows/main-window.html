Settings

This will make this app stay active in the background even when you close all games and this window, to enable Game Time Tracker to scan for processes that might be games, that is not yet supported by Overwolf.

Enabling this setting, will send information in the background when we detect that you're playing a possible game, that we currently do not support.

Enabling this setting, will still track games that Overwolf support, but with our own code instead.

Support

To help us help you out with detection issues and other things, we need you to send us your logs, which can be done by pressing the button below! After you sent the logs, be sure to open a ticket in the Game Time Tracker-Discord (#support channel) that you've done so

The logs method seems to have been disabled, so you can grab logs from %localappdata%\Overwolf\Log, and zip that entire folder and send to me on Discord in a ticket.
